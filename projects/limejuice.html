LimeJuice

Fossil Repository
Web Service
TL;DR: LimeJuice is my own adventure in replicating the great PocketBase using server side Swift instead of Go, and WASM as the integration language instead of Javascript.

As of winter of 2023: LimeJuice is still pretty unfinished, and while the “collections” feature works, most of the interesting WASM integration is still in development.

Screenshots

Technology

  • SvelteKit (Svelte, Typescript, Skeleton UI/Tailwind CSS)
  • Server side Swift with Vapor 4/Swift 5.9
  • JavascriptCore for the JS/WASM engine
  • SQLite
  • WASM SDK in Zig
